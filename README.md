# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

* Two-player game (O vs X)
* Click on box to play
* Box gets disabled after click
* Automatic winner detection
* Simple and clean UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

* Download or clone the project
* Open index.html in browser
* Start playing

## Project Files

* index.html
* style.css
* script.js
